## Cài các gói phụ trợ 
- Chạy lệnh sau để cài các gói phụ trợ 
opencv-python
numpy
matplotlib
SciPy 
uvicorn
fastapi

- chạy: pip install -r requirements.txt
nếu không được thì chạy tường lệnh :

        pip install tên
        
 # Chạy ứng dụng
- Sau khi cài đặt cái gói phụ trợ bạn có thể chạy ứng dụng bằng lênh sau
- taoj data:
        python RecommentData.py
- chay api
        python fastapi_.py
