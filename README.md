# Requirements
Java

Docker, Docker Compose

Nodejs, NPM

Metamask


# Run Application
## Run server
Vào thư mục server

Chạy lệnh
`
    docker compose up
`

Chờ server khởi động (server sẽ tự khởi động nếu xảy ra lỗi)

## Run client
Vào thư mục client

Chạy lệnh
`
    npm start
`

Truy cập trang web ở địa chỉ localhost:3000
